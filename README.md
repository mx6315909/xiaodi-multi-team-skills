# 🤖 小弟多团队协作系统

专业的 AI Agent 团队协作系统，5 个功能团队 + 1 个切换器，覆盖金融、电商、多媒体、办公等场景。

## 📦 团队列表

| 团队 | 图标 | 功能 | 安装命令 |
|------|------|------|----------|
| 金融团队 | 📊 | 股票分析、投资决策 | `clawhub install xiaodi-financial-team` |
| 电商团队 | 🛒 | 选品、运营、定价 | `clawhub install xiaodi-ecom-team` |
| 多媒体团队 | 🎬 | 视频、图片、音频处理 | `clawhub install xiaodi-media-team` |
| 办公团队 | 📋 | 日程、邮件、文档 | `clawhub install xiaodi-office-team` |
| 团队切换器 | 🔄 | 智能切换团队 | `clawhub install xiaodi-team-switcher` |

## ✨ 特性

- **多 Agent 协作**：Master-Worker 架构
- **角色分工明确**：每个团队多个专业角色
- **工作流编排**：支持复杂任务流程
- **环境变量安全**：API Key 通过环境变量配置

## 🚀 快速开始

```bash
# 安装团队
clawhub install xiaodi-financial-team

# 使用示例
分析茅台的投资价值
```
